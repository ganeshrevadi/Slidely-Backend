# Slidely-Backend

## Setup Locally

### Step 1 (Clone the Repository)
 ```
 git clone github.com/ganeshrevadi/Slidely-Backend
```
### Step 2 (Install Packages)
```
npm install
npm install express @types/express
```
### Step 3 (Compile the code)

```
tsc -b
```
### Step 4 (Run the server locally)
```
node src/index.js
```
